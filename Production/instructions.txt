### Blade Knowledgebase: Most Important Components

Box:
- constrains : Versatile layout primitive, supports responsive design, flexbox, styled-system patterns, no custom CSS needed, foundational for layouts
- attributs : as, id, children, flex, flexDirection, flexWrap, flexBasis, flexGrow, flexShrink, display, alignItems, alignSelf, justifyContent, gap, margin, marginTop, marginRight, marginBottom, marginLeft, marginX, marginY, padding, paddingTop, paddingRight, paddingBottom, paddingLeft, paddingX, paddingY, width, height, minWidth, minHeight, maxWidth, maxHeight, backgroundColor, border, borderTop, borderRight, borderBottom, borderLeft, borderColor, borderRadius, borderStyle, borderWidth, position, top, right, bottom, left, zIndex, overflow, overflowX, overflowY, elevation, transform, transformOrigin, clipPath, onMouseOver, onMouseEnter, onMouseLeave, onScroll, onDragStart, onDragEnd, onDragEnter, onDragOver, onDragLeave, onDrop, testID, data-analytics-*

Card:
- constrains : Only accepts  CardBody as children; 
- attributs : children, backgroundColor, borderRadius, elevation, padding, width, height, minHeight, minWidth, maxWidth, isSelected, href, target, rel, accessibilityLabel, shouldScaleOnHover, onHover, onClick, as, testID, data-analytics-*, StyledPropsBlade, paddingBottom, marginBottom, showDivider, title, subtitle, prefix, suffix, visual, paddingTop

TopNav:
- constrains : Top navigation bar, always uses TabNav for multi-product navigation, responsive, integrates with SideNav, only TopNavBrand/TopNavContent/TopNavActions as children, TabNav only accepts TabNavItem as children
- attributs : children, padding, paddingTop, paddingBottom, paddingLeft, paddingRight, paddingX, paddingY, backgroundColor, position, top, bottom, left, right, width, zIndex, testID, StyledPropsBlade, title, href, target, as, isActive, icon, trailing, accessibilityLabel, onMouseDown, onPointerDown, onKeyDown, onKeyUp, onClick

SideNav:
- constrains : Responsive side navigation, only SideNavBody/SideNavFooter as children, supports multi-level, collapsible, mobile, integrates with routing, only SideNavLink/SideNavSection/SideNavItem as children in body
- attributs : children, isOpen, onDismiss, onVisibleLevelChange, banner, testID, StyledPropsBlade, title, description, titleSuffix, trailing, href, target, as, isActive, icon, tooltip, onClick, maxVisibleItems, defaultIsExpanded, onExpandChange, leading, backgroundColor

BottomNav:
- constrains : Persistent mobile navigation, only BottomNavItem as children, positioned at bottom, optimized for touch, supports active state, icons, labels
- attributs : children, zIndex, testID, data-analytics-*, title, icon, href, target, rel, as, isActive, onClick

Breadcrumb:
- constrains : Only BreadcrumbItem as children, shows navigation path, supports size/color, icon, current page, custom component
- attributs : children, size, color, showLastSeparator, testID, data-analytics-*, href, icon, isCurrentPage, accessibilityLabel, onClick, as

Button:
- constrains : Interactive, supports variant/size/color, icon, loading/disabled/fullWidth, can be anchor, accessibility label for icon-only
- attributs : children, variant, color, icon, iconPosition, size, isDisabled, isLoading, isFullWidth, accessibilityLabel, onClick, href, target, rel, type, ref, testID, data-analytics-*, StyledPropsBlade

ButtonGroup:
- constrains : Groups only Button children, consistent styling/spacing, variant/size/color
- attributs : children, variant, size, color, testID, data-analytics-*, StyledPropsBlade

IconButton:
- constrains : Icon-only action, accessible label required, supports size/emphasis/disabled
- attributs : icon, size, emphasis, isDisabled, accessibilityLabel, onClick, testID, data-analytics-*, StyledPropsBlade

Dropdown:
- constrains : Floating overlay, supports various triggers (button, link, input), controlled/uncontrolled, optimal overlay positioning, single/multiple selection, only Dropdown subcomponents as children
- attributs : selectionType, children, isOpen, defaultIsOpen, onOpenChange, shouldCloseOnClickOutside, shouldCloseOnEscape, width, maxWidth, maxHeight, variant, size, isDisabled, icon, iconPosition, title, subtitle, leading, trailing, titleSuffix, onClearButtonClick, label, value, onChange, accessibilityLabel, testID, StyledPropsBlade

Menu:
- constrains : Action menu, not for selection, not responsive by default, trigger and overlay as children, only MenuItem/MenuHeader/MenuFooter/MenuOverlay as children in overlay
- attributs : children, isOpen, onOpenChange, openInteraction, title, description, as, onClick, onFocus, href, target, leading, trailing, titleSuffix, isDisabled, color, zIndex, width, minWidth, maxWidth, subtitle, testID, data-analytics-*

ActionList:
- constrains : List of actionable items, only ActionListItem or ActionListSection as children, can be used in Dropdowns/BottomSheets/SelectInput, supports icons, badges, avatars, sections
- attributs : children, isVirtualized, title, description, onClick, value, href, target, leading, trailing, titleSuffix, isDisabled, intent, isSelected, src, alt, testID, data-analytics-*

TextInput:
- constrains : Single-line text input, supports label, help/error/success text, prefix/suffix, validation, size, controlled/uncontrolled
- attributs : label, name, value, defaultValue, onChange, onFocus, onBlur, placeholder, isDisabled, isRequired, labelPosition, necessityIndicator, helpText, errorText, successText, prefix, suffix, type, autoFocus, size, icon, testID, data-analytics-*

TextArea:
- constrains : Multi-line text input, supports label, help/error/success text, prefix/suffix, validation, size, controlled/uncontrolled
- attributs : label, name, value, defaultValue, onChange, onFocus, onBlur, placeholder, isDisabled, isRequired, labelPosition, necessityIndicator, helpText, errorText, successText, prefix, suffix, autoFocus, size, rows, testID, data-analytics-*
